## Reporting A Vulnerability

Do not create a public GitHub issue for a suspected security vulnerability.
Report it through the AWS vulnerability reporting process:

https://aws.amazon.com/security/vulnerability-reporting/

Include a description of the issue, affected files or resources, reproduction
steps, and any relevant impact assessment.

## Sample Scope

This repository contains a sample application. It is not a production reference
implementation and is not intended to process real financial, customer,
counterparty, account, trade, settlement, or operational data.
